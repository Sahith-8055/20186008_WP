Learning By Doing

1) Each computer has a built-in instruction set that it knows how to execute by design.

Hint

Recall that we use the computation to mean "the execution of a sequence of instructions by a computer with the intention of solving a given problem.

Incorrect. Each computer has a set of primitive instructions that is "wired" to execute. We call this the instruction set of the computer

2) The computer uses intelligence to execute instructions.

Hint
Recall that we said a computer executes the instructions given to it mechanically.
Correct. A computer executes instructions automatically. Its main task is to simply execute the instructions given to it (from its instruction set) and remember their result as the computer goes on. The part that may require intelligence is expressing a corruption in terms of those instructions.
